Frequently Asked Questions
The very first time you access IWM you will need to create you user account by clicking on Create Account located on the login dialog form. Once you have filled in your user info (including others a user and password ) You can then use your newly created user credentials to login into IWM release 4.0
Once you login you will notice on the far right a label entitled sined in as userX userX being your user account ID.
1-Click on menu item userX-Edit Account to change your account info.
2- Click on menu item userX-Change Password to change your password
